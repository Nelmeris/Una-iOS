# Проект: «Una»

Мобильное приложение на операционной системе iOS для сервиса по изучению русского языка как иностранному Una. Назначение: использование мобильного приложения в качестве тренажера и замены веб-сервиса для изучения русского языка студентами, преподавателями или просто людьми, желающими самостоятельно изучить русский язык.
