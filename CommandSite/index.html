Тема проекта

Мобильное приложение на операционной системе iOS для сервиса по изучению русского языка как иностранному Una

Задание

Заказчикам требовался сервис по обучению русскому языку с примером работы хотя бы одного курса по выданной теме с реализацией одного типа решения тренажера Проект актуален в связи с отсутствием качественных сервисов подобной темы в странах СНГ или где-либо еще (со слов заказчика), которым могли бы воспользоваться люди, желающие изучить русский язык, или преподаватели, подбирающие материал и сервисы для обучения своих студентов. Главной задачей данного проекта было создать IOS приложение для сайта Юна, которое сможет работать Offline. В нем должна быть основа для создания и переноса с веб версии тренажеров для обучения и тренировки русского языка, для людей, у которых русский язык не является основным или вообще не изучался. Приложение должно быть понятным, удобным в использовании, приятным глазу и запоминающимся.

Задачи для реализации:

  • Создать "сырые" макеты приложения, продумать их взаимодействие между собой
    (Давтаев Артур)
  • По созданным макетам переделать старый дизайн приложения и придумать новый, более приятный, лаконичный и практичный, для более простого взаимодействия пользователя с приложением
    (Сметанина Александра)
  • Реализовать часть приложения по заданным макетам, продумать архитектуру всего приложения, реализовать его структуру, реализовать взаимосвязь приложения с базой данных и сервером
    (Куфаев Артем)
